# Domaine Multimédia (+intérêt) 

**Graphisme:** l’idée de concevoir des éléments visuels - tels que des images, des textes, des formes et des couleurs - pour transmettre un message ou provoquer une émotion m’attire beaucoup. C’est fou de se dire que ce domaine est partout dans ce que l’on voit chaque jour.

**Animation 2D/3D:** L’animation prend beaucoup de place avec tous les extensions et outils pour raconter des histoires incroyables.

**Illustration:** c’est le parfait monde entre le dessin traditionnel et le numérique, une option qui donne plus de possibilité de rendre son art public.
